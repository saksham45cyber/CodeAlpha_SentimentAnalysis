# Amazon Reviews Sentiment Analysis

## Objective

Analyze customer reviews and classify them into Positive, Negative, and Neutral sentiments.

## Tools Used

* Python
* Pandas
* TextBlob
* Matplotlib
* Seaborn

## Analysis Performed

* Review Processing
* Sentiment Classification
* Sentiment Distribution Visualization

## Key Findings

* Positive reviews indicate customer satisfaction.
* Negative reviews indicate customer dissatisfaction.
* Sentiment analysis helps understand customer opinions.
